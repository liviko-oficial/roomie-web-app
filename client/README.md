# Happy Roomie Frontend

## Quick start

### Pre-requisites

- Have `bun` already install, if you don't have it you can download it at the official [bun site](https://bun.sh/)


### Steps

1. Clone the repository and cd to the server (back end)

    ```shell
    git clone 
    cd rommie-web-app/client
    ```

2. Install dependencies and run project (make sure you have the pre-requisites)

    ```shell
    bun install 
    bun run dev
    ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

> This is a [Next.js](https://nextjs.org) project bootstrapped with [
`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).
> You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
